## Project Name

X Cup SafeBet Co-Signer

## One-Line Description & Project Highlights

A World Cup trading and prediction safety gateway for autonomous agents on X Layer. Every proposed match bet, swap, or payout is converted into a canonical digest; two independent co-signer agents inspect the same digest, and the on-chain gateway executes only when both EIP-712 approvals agree. The demo shows a prompt-injected bad match bet blocked by the sentinel agent and a normal Argentina-France hedge executed with receipt proof.

## Any Other Words To X Layer Team

X Cup SafeBet Co-Signer is designed for the emotional, fast-moving World Cup trading environment. It does not ask judges to trust an AI model's private reasoning: it exposes canonical transaction digests, two independent co-signer verdicts, EIP-712 signatures, and gateway results that can be replayed from the repo.
